# Portfolio — محمود حامد أحمد (محلي)

نسخة محسّنة من البورتفوليو بصيغة **HTML + CSS + JavaScript** فقط — بدون أدوات بناء أو تثبيت.

## ما الذي تم تحسينه؟
- شريط جانبي Drawer يعمل على **كل الشاشات** عبر زر ☰.
- عرض **كل مشاريع GitHub تلقائيًا**.
- إبراز المشاريع **المميزة / ذات النجوم** في الأعلى قبل باقي المشاريع.
- دعم كامل للعربية RTL.
- أقسام أكثر ترتيبًا: نبذة، خبرات، مهارات، مشاريع، تطوع، تواصل.
- تصميم أكثر احترافية وتجربة مستخدم أوضح على الهاتف واللابتوب والشاشات الكبيرة.

## التشغيل
افتح `index.html` مباشرة في المتصفح، أو شغّل سيرفر محلي:

```bash
python3 -m http.server 8000
```

ثم افتح:

```text
http://localhost:8000
```
